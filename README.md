# 📖 Project Overview

**Live Stock Tracker** is a Python-based automation project developed using **Playwright**. The application automates browser interactions to retrieve live stock market information, processes the extracted data, and saves it into a CSV file.

This project demonstrates practical experience with:

* 🐍 Python Programming
* 🎭 Browser Automation using Playwright
* ⚡ Asynchronous Programming (`asyncio`)
* 📄 CSV File Handling
* 🧩 Object-Oriented Programming (OOP)
* 📊 Live Data Extraction

# ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Live-Stock-Tracker.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Live-Stock-Tracker
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Install Playwright Browsers

```bash
playwright install
```

---

# ▶️ Run the Project

```bash
python main.py
```
